AI Prompt Engineering – ColdStack Protocol v0.1

A tactical micro-course for using AI as a cold copilot.

Day 1: Foundations

  • Prompt = Intent + Frame + Constraints
  • Intent: What you want
  • Frame: How you want it (format/style/tone)
  • Constraints: Word count, style rules, tools used
  • Example: "Write a Zellij config with 3 panes in TOML with comments."

Day 2: Tactical Prompts

  • System Drafting: "Build me a system for [X goal] in [Y time] using [Z evidence]."
  • Explainer Mode: "Explain [tool] to a beginner who only knows [context]."
  • Code Companion: "Write a shell script that does [task]. Output only code with inline comments."
  • Compare + Decide: "Compare A vs B for [goal]. Bullet list pros/cons. Recommend one."

Day 3: Iteration & Debugging

  • Ask: What was wrong with the output? What should it have focused more/less on? What context was missing?
  • Prompt: "Regenerate but make it shorter / simpler / more tactical."

Day 4: Compression & Expansion

  • Summary: "TL;DR this into 3 bullets."
  • Depth: "Expand this 3-point summary into a full document."

Day 5+: ColdStack Integration

  • Draft logs, skills, sprints
  • Generate or refine code tools
  • Write scoped SOPs (Standard Ops Procedures)
  • Translate abstract thoughts into buildable tasks
